# Student Attendance & Timetable Management Dashboard
A front-end web application developed to track personal attendance across subjects.
The dashboard allows students to monitor attendance percentage and identify subjects
with low attendance.

## Features
- Subject-wise attendance tracking
- Automatic percentage calculation
- Visual alerts for low attendance
- Persistent storage using browser LocalStorage
- Clean and responsive UI

## Technologies
- HTML
- CSS
- JavaScript (DOM, LocalStorage)
